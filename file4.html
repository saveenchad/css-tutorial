Example

Border Spacing

Usage

border-spacing: length | initial | inherit;

The spacing between borders for elements with multiple borders

Note: if [x] is given for length, then spacing is set as [x] all the way around.

Note: if [x y] is given for length, [x] is the horizontal spacing and [y] is the vertical spacing.

Example

border-spacing: px
Row 1, Col 1 Row 1, Col 2
Row 2, Col 1 Row 2, Col 2
