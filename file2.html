Border Bottom Color

Usage

border-bottom-color: color | transparent | initial | inherit;

This property modifies the color of the bottom part of the border.

Example

color:

Border Bottom Style

Usage

border-bottom-style: none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset | initial | inherit;

This property modifies the style of the bottom part of the border.

Note: hidden is not supported in IE7 and earlier

Example

border-bottom-style: none hidden dotted dashed solid double groove ridge inset outset initial
