Text Decoration

Usage

text-decoration: none | underline | overline | line-through | initial | inherit

The text-decoration property specifies the decoration added to text.

Example

text-decoration: none underline overline line-through initial inherit

Text Decoration Color

Usage

text-decoration-color: color | initial | inherit

The text-decoration-color property specifies the color of the text-decoration (underlines, overlines, linethroughs).

Note: The text-decoration-color property will only have an effect on elements with a visible text-decoration.

Note: The text-decoration-color property is not supported in any of the major browsers.

Example

text-decoration-color: black red purple

Property Name

Usage

text-indent: length | initial | inherit

The text-indent property specifies the indentation of the first line in a text-block.

Example

text-indent: 20px 50px 100px

Transform

Usage

transform: none | transform-functions | initial | inherit

The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

Example

transform: none matrix(0.866,0.5,-0.5,0.866,0,0) rotate(10deg) skewY(20deg)

Transform Origin

Usage

transform-origin: x-axis y-axis z-axis | initial | inherit

The transform-origin property allows you to change the position of transformed elements.

Note: This property must be used together with the transform property.

Example

These radio buttons affect the previous div ^^.
transform-origin: initial rotate(45deg) 20% 40%;

Transform Style

Usage

transform-style: flat | preserve-3d | initial | inherit

The transform-style property specifies how nested elements are rendered in 3D space.

Note: This property must be used together with the transform property.

Example

transform-style: preserve-3d

Transition Duration

Usage

Transition Timing Function

Usage

transition-timing-function: linear | ease | ease-in | ease-out | ease-in-out | step-start | step-end | steps(int,start|end) | cubic-bezier(n,n,n,n) | initial | inherit

The transition-timing-function property specifies the speed curve of the transition effect.

This property allows a transition effect to change speed over its duration.

Example

transition-timing-function: linear ease ease-in ease-out ease-in-out

Left

Usage

left: auto | length | initial | inherit

For absolutely positioned elements, the left property sets the left edge of an element to a unit to the left/right of the left edge of its nearest positioned ancestor.

Note: If an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.

Example

left: auto 10px 50px 100px

Perspective

Usage

perspective: length | none

The perspective property defines how many pixels a 3D element is placed from the view. This property allows you to change the perspective on how 3D elements are viewed.

When defining the perspective property for an element, it is the CHILD elements that get the perspective view, NOT the element itself.

Example

perspective: 200px

Perspective Origin

Usage

perspective-origin: x-axis y-axis | initial | inherit

The perspective-origin property defines where a 3D element is based in the x- and the y-axis. This property allows you to change the bottom position of 3D elements.

When defining the perspective-origin property for an element, it is the CHILD elements that are positioned, NOT the element itself.

Example

perspective-origin: 10% 10%
