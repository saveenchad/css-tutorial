Font Size Adjust

Usage

font-size-adjust: number | none | initial | inherit;

This property allows the programmer to set the font size in case a secondary font is needed from a font-family.

This property is only supported in Firefox!

Example

font-size-adjust: px
this is a font-size-adjust test!
