Font Size Adjust

Usage

font-size-adjust: number | none | initial | inherit;

This property allows the programmer to set the font size in case a secondary font is needed from a font-family.

This property is only supported in Firefox!

Example

.times { font-family: 'Times New Roman'; font-size-adjust: 0.58; }
.verdana { font-family: 'Verdana'; font-size-adjust: 0.58; }
<p class="times">this is a font-size-adjust test!</p>
<p class="verdana">I have the same font size as the paragraph above even though I'm a different font!</p>

this is a font-size-adjust test!

I have the same font size as the paragraph above even though I'm a different font!
