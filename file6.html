Clip

Usage

clip: auto | shape | initial | inherit

What happens if an image is larger than its containing element? - The clip property lets you specify a rectangle to clip an absolutely positioned element. The rectangle is specified as four coordinates, all from the top-left corner of the element to be clipped.

Example

clip: auto rect(0px,60px,200px,0px) initial inherit

Animation Direction

Usage

animation-direction: normal | reverse | alternate | alternate-reverse | initial | inherit;

The animation-direction property defines whether an animation should play in reverse direction or in alternate cycles.

Note: The animation-direction property is not supported in Internet Explorer 9 and earlier versions.

Note: The "reverse" and "alternate-reverse" values are not supported Safari.

Example

animation-direction: normal reverse alternate alternate-reverse

Animation Duration

Usage

animation-duration: time | initial | inherit

The animation-duration property defines how many seconds or milliseconds an animation takes to complete one cycle.

Note: The animation-duration property is not supported in Internet Explorer 9 and earlier versions.

Example

animation-duration: 500ms 1s 2s 5s

Animation Play State

Animation Timing Function

Usage

animation-timing-function: linear | ease | ease-in | ease-out | ease-in-out | step-start | step-end | steps(int,start|end) | cubic-bezier(n,n,n,n) | initial | inherit

The animation-timing-function specifies the speed curve of an animation.

The speed curve defines the TIME an animation uses to change from one set of CSS styles to another.

The speed curve is used to make the changes smoothly.

Example

animation-timing-function: linear ease ease-in ease-out ease-in-out
