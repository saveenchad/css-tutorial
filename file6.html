Example

columns: auto 100px 4 150px 3 initial inherit

Column Width

Usage

column-width: auto | length | initial | inherit;

The column-width property specifies a suggested, optimal width for the columns.

Note: The column-width property is not supported in Internet Explorer 9 and earlier versions.

Example

column-width: auto 100px 150px 400px initial inherit

Example

Column Gap

Usage

column-gap: length | normal | initial | inherit

The column-gap property specifies the gap between the columns.

Note: If there is a column-rule between columns, it will appear in the middle of the gap.

Example

column-gap: normal 10px 50px initial inherit
